NAME

Pod::Site - Create websites from pod

SYNOPSIS

  use Pod::Site;

  my $site = Pod::Site->new( dir => '/path/to/site/ );

  $site->install_file( 'path/to/Foo/Bar.pm', 'Foo/Bar.html' );

  $site->install_dir( 'lib' );

DESCRIPTION

Pod::Site helps you create a web site of documentation.

Most of the work is done by Pod::POM . Pod::POM takes care of parsing the pod and creating the html output. Pod::Site manages what pod files to parse and where to store the output.
